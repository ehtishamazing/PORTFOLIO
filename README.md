# Ehtisham Ul Haq — Portfolio

## How to use it tomorrow
1. Open `index.html` in any browser (double-click it, no install needed) — that's the whole site, works fully offline except the Google Fonts and the Instagram/LinkedIn/Substack links.
2. To show it on a laptop in the interview: just open the file. To send it as a link: upload the whole `portfolio` folder to Netlify Drop (netlify.com/drop) or Vercel and you'll get a live URL in under a minute — keep `assets/` in the same folder as `index.html` when you do.
3. Don't rename `index.html` or move it out of this folder, or the photos will stop loading (they're linked as `assets/headshot.jpg` etc.).

## What's inside
- `index.html` — the whole site: hero, on-camera reel links, featured Dawn column + two poems, hosting record, client SEO work, CSSPrep/Researchers Avenue, contact.
- `assets/` — your three photos, compressed for web.

## If you want to edit anything
- All the text lives directly in `index.html` — search for the line you want to change and edit it, no build step required.
- Colors and fonts are set as CSS variables near the top of the `<style>` block (`--ink`, `--press`, `--brass`, etc.) if you want to retheme.
